# MorphCam — Releases

This repository hosts the official Windows installer for **MorphCam**.

MorphCam is a real-time AI camera app for Windows that transforms your
face into an AI persona for streaming, video calls, and content creation.

## Download

Get the latest installer from the [**Releases**](../../releases/latest) page.

- **Platform:** Windows 10 / 11 (64-bit)
- **File:** `MorphCam_<version>_x64-setup.exe`

Run the installer and follow the prompts. For setup help, visit
[usemorphcam.com/download](https://usemorphcam.com/download).
